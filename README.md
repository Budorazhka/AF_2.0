# Клубный дом — лендинг

Премиальный одностраничный сайт клубного дома класса de-luxe.
Скролл-сторителлинг по «главам», золото-графитовая палитра, серифная типографика.

Стек: статический сайт (HTML + CSS + vanilla JS), отдаётся через **nginx** в Docker
с gzip и кэшированием. Без сборки и зависимостей — максимально быстро грузится.

## Структура
```
index.html        — все секции (главы) лендинга
css/styles.css    — стили, палитра, анимации
js/main.js        — скролл-анимации, навигация, lazy-load, форма
nginx.conf        — gzip + кэш + security-заголовки
Dockerfile        — образ на nginx:alpine
docker-compose.yml
.env              — порт и контактные данные (WEB_PORT=8090)
assets/           — фото/видео (см. assets/README.md)
```

## Запуск
```bash
docker compose up -d --build
```
Сайт: `http://<server-ip>:8090` (порт меняется через `WEB_PORT` в `.env`).

## Медиа
Имена файлов уже зашиты в разметку — кладите в `assets/img` и `assets/video`
(подробно: `assets/README.md`). Папка примонтирована как volume, пересборка не нужна.

## Что доделать под клиента
- Заменить тексты-плейсхолдеры и имена авторов на реальные.
- Подставить телефон/почту/адрес (в `.env` и в `index.html`).
- Подключить отправку формы (Telegram-бот или SMTP) — место помечено `TODO` в `js/main.js`.
- Завести домен и проксировать на порт 8090 через общий nginx.
