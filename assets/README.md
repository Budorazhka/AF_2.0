# Медиа-материалы

Сюда складываются фото и видео. Имена файлов уже прописаны в `index.html` —
просто положите файлы с такими именами, и они подхватятся автоматически
(пока файла нет, в блоке виден серый плейсхолдер с подписью, что нужно положить).

## Видео
- `video/hero.mp4` — фоновое видео на первом экране (рекомендуется 1920×1080, H.264, без звука, до ~8–12 МБ)

## Сжатие (чтобы грузилось быстро)
Рекомендую перегнать в WebP/оптимизированный JPG. Пример:

```bash
# JPG → оптимизация (нужен imagemagick)
mogrify -resize '1920x1920>' -quality 82 -strip img/*.jpg

# Видео → лёгкий mp4 (нужен ffmpeg)
ffmpeg -i raw.mov -vf "scale=1920:-2" -c:v libx264 -crf 26 -preset slow -an video/hero.mp4
```

Контейнер монтирует папку `assets/` как volume — после добавления файлов
**пересборка не нужна**, просто обновите страницу.
